What is a Variable?


A variable is a named storage location in memory where you can store data (like

numbers, characters, etc.) temporarily while your program runs.


Example

           int age = 25;

    Types of Variables (Based on Data Types)


    | Data Type | Description                  | Example                |
| --------- | ---------------------------- | ---------------------- |
| `int`     | Integer (whole number)       | `int age = 21;`        |
| `float`   | Decimal number               | `float rate = 5.5;`    |
| `char`    | Single character             | `char grade = 'A';`    |
| `double`  | Large decimal number         | `double pi = 3.14159;` |
| `void`    | No value (used in functions) | `void function()`      |


    Simple Example Program with Explanation


         #include < stdio.h>

   int main() {
    int a = 10;              // Line 1: declares variable a and stores 10
    int b = 20;              // Line 2: declares variable b and stores 20
    int sum = a + b;         // Line 3: adds a and b, stores result in sum
    printf("Sum = %d", sum); // Line 4: prints the value of sum using printf
    return 0;                // Line 5: ends the program
}


    output
    

Sum = 30
