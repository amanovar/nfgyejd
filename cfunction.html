Function ek aisa block hota hai jo specific task perform karta hai. Isse code modular, reusable aur manageable ban jata hai.
English: A function is a block of code that performs a specific task. Functions help in reusability and code organization.

Example:

ðŸ“š Types of Functions in C


  • 1. Library Functions: Inbuilt functions jaise printf(), scanf(), sqrt().
  • 2. User-defined Functions: Jo hum khud banate hain specific task ke liye.

ðŸ“Œ User-defined Functions ke 4 Types:


  1. Function with no argument and no return value
  2. Function with argument but no return value

  4. Function with no argument but return value
