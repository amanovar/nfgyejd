Introduction


In C programming, keywords are reserved words that have special meaning to the compiler. They cannot be used as identifiers (e.g., variable names, function names). Keywords define the structure and behavior of the program. C language supports 32 keywords such as int, for, if, etc.




Types of Keywords


Keywords in C can be categorized based on their usage:


  • Data Type Keywords: Define the type of data a variable can hold. Examples: int, char, float, double.

  • Control Flow Keywords: Control the flow of execution in a program. Examples: if, else, switch, case, default, break, continue.

  • Looping Keywords: Used to create loops. Examples: for, while, do.

  • Function Keywords: Define functions and their return types. Examples: return, void.

  • Storage Class Keywords: Define the scope and lifetime of variables. Examples: auto, static, extern, register.

  • Other Keywords: Serve various purposes. Examples: sizeof, const, typedef, volatile, goto, enum, struct, union, signed, unsigned.

Examples


1. Data Type Keywords


        #include 

        int main() {
          int age = 25;
          char grade = 'A';
          float height = 5.9;
          double pi = 3.14159;

          printf("Age: %d\n", age);
          printf("Grade: %c\n", grade);
          printf("Height: %.1f\n", height);
          printf("Pi: %.5f\n", pi);

          return 0;
        }


2. Control Flow Keywords


        #include 

        int main() {
          int score = 85;

          if (score >= 90) {
            printf("Grade: A\n");
          } else if (score >= 80) {
            printf("Grade: B\n");
          } else {
            printf("Grade: C\n");
          }

          return 0;
        }


3. Looping Keywords


        #include 

        int main() {
          for (int i = 1; i <= 5; i++) {
            printf("Iteration %d\n", i);
          }

          return 0;
        }


4. Function Keywords


        #include 

        void greet() {
          printf("Hello, World!\n");
        }

        int main() {
          greet();
          return 0;
        }


5. Storage Class Keywords


        #include 

        void example() {
          static int count = 0;
          printf("Count: %d\n", count);
          count++;
        }

        int main() {
          example();
          example();
          return 0;
        }
