üîπ Array Kya Hai? (What is an Array?)


Array ek aisi data structure hai jo same type ke multiple values ko ek hi variable ke andar store karti hai.
English: An array is a collection of elements of the same data type stored at contiguous memory locations.


üîß Array Banane Ka Syntax:


data_type array_name[size];

Example:

int marks[5];

üìå Array Ko Initialize Kaise Karein?


int numbers[5] = {10, 20, 30, 40, 50};

üîÅ Array Aur Loop ka Use (Traversal Example):


#include <stdio.h>

int main() {
    int i;
    int numbers[5] = {10, 20, 30, 40, 50};

    for(i = 0; i < 5; i++) {
        printf("Element %d: %d\n", i, numbers[i]);
    }

    return 0;
}

üßæ Output:

Element 0: 10
Element 1: 20
Element 2: 30
Element 3: 40
Element 4: 50

üìö Types of Arrays in C


  • 1. One-Dimensional Array
    Simple linear array:
    int arr[10];

  • 2. Two-Dimensional Array
    Table/matrix format:
    int matrix[3][3];

  • 3. Multi-Dimensional Array
    Nested arrays (rare use):
    int space[2][3][4];

  • 4. String as Character Array
    char name[10] = "Amit";


üí° 2D Array Example (Matrix):

#include <stdio.h>

int main() {
    int i, j;
    int matrix[2][2] = {{1, 2}, {3, 4}};

    for(i = 0; i < 2; i++) {
        for(j = 0; j < 2; j++) {
            printf("%d ", matrix[i][j]);
        }
        printf("\n");
    }

    return 0;
}

üßæ Output:

1 2
3 4
