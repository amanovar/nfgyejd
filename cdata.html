Introduction to Data Types in C


  • In C language, data types tell the compiler what kind of data a variable will store.
  • This helps the computer understand how much memory to reserve and how to interpret the data.


    • Examples of data:


  • Whole numbers → int

  • Decimal numbers → float,double

  • Single characters → char


    • Types of Data Types in C


    Range: Typically -32,768 to 32,767 (on a 16-bit system)


            
    Example
    
        
    
        int age = 25;


    float – For decimal (floating point) numbers


            
    Example
    
        
    
        float price = 99.50;


    char – For single characters


            
    Example
    
        
    
        char grade = 'A';


    double – For more precise decimal numbers (greater precision than float)


            
    Example
    
        
    
        double pi = 3.1415926535;


    Other Modifiers


    You can modify data types to increase range or change behavior:

    short long unsigned


            
    Example
    
        
    
        unsigned int x = 100;  // Only stores positive integers


    Example Using Different Data Types


            #include < stdio.h>

   int main() {
    int age = 22;
    float weight = 65.5;
    char grade = 'B';
    double pi = 3.14159;

    printf("Age: %d\n", age);
    printf("Weight: %.2f\n", weight);
    printf("Grade: %c\n", grade);
    printf("Value of Pi: %.5lf\n", pi);

    return 0;
}

    

    output
    

    
Age: 22
Weight: 65.50
Grade: B
Value of Pi: 3.14159
