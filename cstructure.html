ðŸ”¹ Structure Kya Hai? (What is a Structure?)


Structure ek user-defined data type hai jisme hum alag-alag types ke variables ko ek saath store kar sakte hain.
English: A structure in C is a user-defined data type that allows grouping of variables of different types under one name.


ðŸ”§ Structure Banane Ka Syntax:


struct structure_name {
    data_type member1;
    data_type member2;
    ...
};

Example:

struct Student {
    int roll;
    char name[50];
    float marks;
};

ðŸ§ª Structure Variable Declare Karna:


struct Student s1;

ðŸ“Œ Members Ko Access Karna (Dot Operator):


s1.roll = 101;
strcpy(s1.name, "Amit");
s1.marks = 89.5;

ðŸ§ª Full Example:


#include <stdio.h>
#include <string.h>

struct Student {
    int roll;
    char name[50];
    float marks;
};

int main() {
    struct Student s1;

    s1.roll = 1;
    strcpy(s1.name, "Rahul");
    s1.marks = 95.2;

    printf("Roll: %d\n", s1.roll);
    printf("Name: %s\n", s1.name);
    printf("Marks: %.2f\n", s1.marks);

    return 0;
}

ðŸ§¾ Output:

Roll: 1
Name: Rahul
Marks: 95.20

ðŸ“š Structure Ke Features


  • Multiple data types ko ek group me rakh sakte hain
  • Code more organized aur real-world model ke jaise ban jata hai
  • Useful in arrays of structure, functions ke sath use, pointer to structure

ðŸ“Œ Array of Structures Example:


#include <stdio.h>

struct Employee {
    int id;
    float salary;
};

int main() {
    struct Employee e[2] = {{1, 5000}, {2, 7000}};
    
    for(int i = 0; i < 2; i++) {
        printf("ID: %d, Salary: %.2f\n", e[i].id, e[i].salary);
    }

    return 0;
}
