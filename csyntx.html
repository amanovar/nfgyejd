Syntax


You have already seen the following code a couple of times in the first chapters. Let's break it down to understand it better:



Basic Syntax Rules

| Rule                           | Example                                |
| ------------------------------ | -------------------------------------- |
| Each statement ends with `;`   | `int a = 5;`                           |
| Functions are enclosed in `{}` | `void func() { ... }`                  |
| Case sensitive                 | `Var` ≠ `var`                          |
| Comments:                      | `// single line` or `/* multi-line */` |

Example

        #include 

int main() {
  printf("Hello World!");
  return 0;
}

Line-by-line


  • #include < stdio.h>

    • ➤ Yeh ek preprocessor directive hai.stdio.h ka matlab "Standard Input Output" – isse hum printf() aur scanf() jaisi cheezein use kar sakte hain.


  • int main()

    • ➤ Main function se hi C program start hota hai.

    int ka matlab function integer return karega


  • printf("Hello World");

    • ➤ Yeh output line hai. Console pe "Hello World" print karega.

    printf() ek function hai stdio.hse.


  • return 0;

    • ➤ Yeh batata hai ki program successfully run hua.

    0 return karke OS ko success ka signal deta hai.



    Example Program

            #include 

    int main() {
    int a = 5;                  // integer variable a banaya aur value 5 di
    int b = 10;                 // integer variable b banaya aur value 10 di
    int sum = a + b;           // sum naam ka variable, jisme a aur b ka addition
    printf("Sum = %d", sum);   // result print kara, %d ka matlab integer value
    return 0;
}



    output
