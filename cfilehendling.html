üîπ File Handling Kya Hai?


File handling ka use hum tab karte hain jab data ko permanent store karna hota hai.
English: File handling allows a C program to read from and write to files, making data permanent beyond program execution.


üìå File Handling Ke Functions (stdio.h):


  • fopen() ‚Äì file open karne ke liye
  • fclose() ‚Äì file close karne ke liye
  • fprintf(), fscanf() ‚Äì file me write/read karne ke liye (formatted)
  • fgets(), fputs() ‚Äì string ke liye
  • fread(), fwrite() ‚Äì binary files ke liye

üìÇ File Modes:


  • "r" ‚Äì Read
  • "w" ‚Äì Write (overwrite)
  • "a" ‚Äì Append
  • "r+", "w+", "a+" ‚Äì Read/Write combinations

üìù Example: Write to a File


#include <stdio.h>

int main() {
    FILE *fp;
    fp = fopen("file.txt", "w");

    if (fp == NULL) {
        printf("File not found!\n");
        return 1;
    }

    fprintf(fp, "Hello, this is a test.\n");
    fclose(fp);

    printf("Data written successfully.\n");
    return 0;
}

üßæ Output:

Data written successfully.


üìñ Example: Read from a File


#include <stdio.h>

int main() {
    FILE *fp;
    char ch;

    fp = fopen("file.txt", "r");

    if (fp == NULL) {
        printf("File not found!\n");
        return 1;
    }

    while ((ch = fgetc(fp)) != EOF) {
        putchar(ch);
    }

    fclose(fp);
    return 0;
}

üßæ Output (depends on file content):


Hello, this is a test.

üìö Tips for File Handling


  • File pointer ka type hamesha FILE * hota hai
  • fopen() return karega NULL agar file open nahi hui
  • Always use fclose() after operations
  • File operations me error checking zaroori hai


üìå Append Data to File:


FILE *fp = fopen("file.txt", "a");
fprintf(fp, "More data...\n");

üìå File handling real-world applications jaise data storage, logging, user records, report generation me kaam aati hai.
