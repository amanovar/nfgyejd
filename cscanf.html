What is scanf() in C?


scanf() is an input function in C used to read data from the user (keyboard) and store it into variables.


It is part of the < stdio.h> (Standard Input Output) header file.



Syntax of scanf()

            scanf("format_specifier", &variable);

üîπ Explanation:


  • "format_specifier" Tells the type of data to be read (%d, %f,etc.)

    • &variable: The address of the variable where the input will be stored (& is the "address-of" operator)



    Example: Taking Integer Input


    #include < stdio.h>

int main() {
    int age;
    printf("Enter your age: ");
    scanf("%d", &age);               // %d for integer, &age gives address
    printf("You are %d years old.", age);
    return 0;
}



    Output:
    

Enter your age: 18
You are 18 years old.


    Format Specifiers for scanf()


  • These are used to match the type of input:

    • 
| Data Type             | Format Specifier | Example Input | Example Code            |
| --------------------- | ---------------- | ------------- | ----------------------- |
| `int`                 | `%d`             | `23`          | `scanf("%d", &num);`    |
| `float`               | `%f`             | `3.14`        | `scanf("%f", &pi);`     |
| `double`              | `%lf`            | `123.456`     | `scanf("%lf", &x);`     |
| `char`                | `%c`             | `A`           | `scanf(" %c", &grade);` |
| `string` (char array) | `%s`             | `hello`       | `scanf("%s", name);`    |

    ‚ö†Ô∏è Note: For characters, always use " %c" with a space before %c to avoid newline problems.
