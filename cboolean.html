In programming, a Boolean data type represents one of two possible values: true or false. These are used to control the flow of a program based on conditions.


Example:

#include <stdio.h>
#include <stdbool.h>

int main() {
    bool isRaining = true;
    if (isRaining) {
        printf("It's raining.\n");
    } else {
        printf("It's not raining.\n");
    }
    return 0;
}

Output:

It's raining.
