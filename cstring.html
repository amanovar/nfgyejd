In C programming, a string is a sequence of characters terminated with a null character '\0'. Unlike many other programming languages, C does not have a built-in string type. Instead, strings are represented as arrays of characters.



Types of C Strings


There are two primary ways to declare strings in C:

  • Character Array: char str[10];
  • String Literal: char *str = "Hello";


Examples


Declaration and Initialization


        char greeting[] = "Hello, World!";
        char name[20] = "Alice";


Accessing Characters


        printf("%c", greeting[0]); // Outputs: H


Modifying Strings


        greeting[0] = 'J';
        printf("%s", greeting); // Outputs: Jello, World!


Common String Functions

  • strlen(str) - Returns the length of the string str.
  • strcpy(dest, src) - Copies the string src to dest.
  • strcmp(str1, str2) - Compares two strings str1 and str2.
  • strcat(dest, src) - Appends the string src to the end of dest.
